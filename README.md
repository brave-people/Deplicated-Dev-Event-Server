## 기능

- 인증
  - [v] 로그인
    - [v] JWT 토큰 발급
  - [] 로그아웃
  - [v] 회원가입
    - [v] 비밀번호 암호화
- 프로필
  - [v] 사용자 정보 불러오기
  - [v] 사용자 정보 수정
- 관리자
  - [] 이벤트 전체보기
  - [] 이벤트 추가
  - [] 이벤트 수정
  - [] 이벤트 삭제
- 행사 목록
  - [] 행사 목록 조회
- 행사 요청
  - [] 글쓰기
  - [] 글수정
